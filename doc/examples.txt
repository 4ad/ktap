
KTAP Script Examples
--------------------

1. syscall tracing example

	trace "syscalls:*" function (e) {
		print(cpu(), pid(), execname(), e)
	}

2. timer example
	timer.s(1, function () {
		print("1s timer hit")
	})

	os.wait()

3. kprobe example:
	trace "probe:schedule" function (e) {
		print("kprobe schedule hit")
	})
